Robotron 2084 Remake

Project Details

This is a recreation of the classic game Robotron 2084. Created in my own game engine.

Robotron 2084

This was my first project where I really made a proper game. It was an incredible learning experience for me!

I made this early on during the development of my engine, and the process is what inspired me to redesign my engine using ECS. With the design decisions I was required to make, the benefit of decoupling the data from the systems logic became very apparent.

I am currently working on porting this to the most recent version of my engine so I can utilize all the features it now offers.
